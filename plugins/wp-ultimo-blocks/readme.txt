WP Ultimo: Blocks

Use WP Ultimo blocks on the new WordPress block editor!

== Installation ==

1. Upload 'wp-ultimo-blocks' to the '/wp-content/plugins/' directory
2. Activate the plugin through the 'Plugins' menu in your WordPress Network Admin
