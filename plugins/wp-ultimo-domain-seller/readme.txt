== Description ==

Plugin Name: WP Ultimo: Domain Seller (FP)

Feature Plugin containing the domain selling capabilities, implementing OpenSRS first

== Installation ==

1. Upload 'wp-ultimo-domain-seller' to the '/wp-content/plugins/' directory
2. Activate the plugin through the 'Plugins' menu in your WordPress Network Admin
